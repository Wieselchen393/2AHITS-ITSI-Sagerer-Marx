# Grundlagen

Shell: ist ein Programm das eine textorientierte Schnittstelle zum Betriebssystem zur Verfügung stellt.

```sh
~/workspace$
```

Einfacher Befehl:

```sh
$ date
```

Jeder Befehl ht Optionen über die das Verhalten gesteuert werden kann.

```sh
$ date -I 
```

## Optionen:
- short-hand = 1 Buchstabe
- long-hand = Wort

```sh
$ date --iso-8601
```

Quelle zu Informationen über einen Befehl:

- häufig: Option `--help`
- man pages (Manual = Handbuch) `$ man date`,
  nicht in Replit aber in gängigen Linux Distributionen
- Internetsuche mit "man date"
- KI

## Argumente

Daten mit den ein Befehl arbeitet
```sh
$ echo Hallo Welt
```
`Hallo Welt` ist hier das Argument

mit Optionen
```sh
$ echo -n Hallo Welt
```
## Tastenkürzel

- Pfeil nach oben/unten: Historie
- Tabulator: automatische File Vervollständigung
- Strg-A: Anfang der Zeile
- Strg-E: Ende der Zeile
- Strg-C: Abbruch des aktuellen Kommandos

## File Befehle

- `cd`: Wechselt zu einen anderen Arbeitsverzeichniss (working Directory), cd .. geht ein Verzeichnis nach oben, ohne Pfad/Argument --> Homeverzeichnis
- `mkdir`: erstellt ein Unterverzeichnis im Arbeitsverzeichnis
- `ls`: listet den Inhalt des working Directorys auf, -l ausführliche Informationen über die Datein im Verzeichnis, -a zeigt versteckte Datein an welche mit einen Punkt vor dem Dateinamen     definiert sind
- `touch`: legt eine neue Datei mit den angegebenen Namen an
- `pwd`: gibt den absoluten Pfad des aktuellen Verzeichnis an, -l gibt den logische Pfad auch an wenn sysmbolische Links existieren
- `rm`: löscht die angegebene Datei, -r löscht Verzeichnise und Unterverzeichnise sowie alle Datein darin
- `rmdir`: löscht das angegebene Verzeichnis (Löscht nur leere Verzeichnise)

## Unterschied absoluter und relativer Pfad

- absoluter Pfad: Pfad zur Datei vom Stammverzeichnis
- relativer Pfad: Pfad zu Datei vom aktuellen Arbeitsverzeichnis


## Bedeutung von cd (ohne Argument):
- Kehrt zum Homeverzeichnis des Benutzers zurück

## Bedutung von ~ :
- wird genutzt um das Homeverzeichnis des Benutzers klar zu markieren
